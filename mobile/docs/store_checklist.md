# App Store Checklist

## Pre-Launch Checklist

### Android (Google Play Store)

- [ ] App icon (1024x1024px PNG)
- [ ] Feature graphic (1024x500px PNG)
- [ ] Screenshots (phone and tablet)
  - [ ] At least 2 phone screenshots
  - [ ] At least 2 tablet screenshots (if tablet support)
- [ ] App description (4000 characters max)
- [ ] Short description (80 characters max)
- [ ] Privacy policy URL
- [ ] Content rating questionnaire
- [ ] Target audience
- [ ] Category selection
- [ ] Contact email
- [ ] Store listing graphic
- [ ] Promotional text (optional)
- [ ] Application signing key configured
- [ ] Version code incremented
- [ ] Minimum SDK version set
- [ ] ProGuard rules configured (if enabled)
- [ ] App bundle built and signed

### iOS (App Store)

- [ ] App icon (1024x1024px PNG)
- [ ] Screenshots (all required device sizes)
  - [ ] iPhone 6.7" display
  - [ ] iPhone 6.5" display
  - [ ] iPhone 5.5" display
  - [ ] iPad Pro 12.9" (if tablet support)
- [ ] App description (4000 characters max)
- [ ] Subtitle (30 characters max)
- [ ] Keywords (100 characters max)
- [ ] Support URL
- [ ] Marketing URL (optional)
- [ ] Privacy policy URL (required if app collects data)
- [ ] Category selection
- [ ] Age rating
- [ ] App review contact information
- [ ] Export compliance
- [ ] Content rights
- [ ] Advertising identifier usage
- [ ] App Store Connect app configured
- [ ] Bundle ID registered
- [ ] Provisioning profiles set up
- [ ] Code signing configured
- [ ] IPA built and signed

### General Requirements

- [ ] App name finalized
- [ ] Package name/bundle ID confirmed
- [ ] Version number set
- [ ] Build number set
- [ ] Permissions justified and documented
- [ ] Privacy policy created and hosted
- [ ] Terms of service created (if applicable)
- [ ] Support email configured
- [ ] App tested on multiple devices
- [ ] Performance optimized
- [ ] Memory leaks checked
- [ ] Crash reporting configured
- [ ] Analytics configured
- [ ] Release notes prepared

### Content Guidelines

- [ ] No offensive content
- [ ] Content rating appropriate
- [ ] User-generated content moderated
- [ ] Intellectual property rights cleared
- [ ] No misleading information
- [ ] Clear value proposition

### Technical Requirements

- [ ] App size optimized (< 100MB recommended)
- [ ] Startup time optimized
- [ ] Battery usage reasonable
- [ ] Network usage optimized
- [ ] No hardcoded secrets/keys
- [ ] Error handling comprehensive
- [ ] Offline capability (if applicable)
- [ ] Backward compatibility considered
